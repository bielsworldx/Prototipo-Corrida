
# Protótipo de Corrida 2D

Protótipo simples de um jogo de corrida singleplayer.


## Aprendizados

- Utilização de retângulos
- Funções com eventos

## Funcionalidades

- Movimento com teclas direcionais
- Vitória com pop-ups



## Stack utilizada

**Front-end:** HTML, CSS, JavaScript
